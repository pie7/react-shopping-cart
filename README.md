## Overview

Simple Shopping Cart with Product Page

### `Features`

- Product Page
- Cart Page
- Add/Update/Delete Product
- Search Product

### `Web APIs`

- [Pixabay](https://pixabay.com/)


### `Tools`

- React-Route
- Redux
- Styled Components
